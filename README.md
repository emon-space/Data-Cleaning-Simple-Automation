# Data-Cleaning-Simple-Automation
Python scripts and notebooks for cleaning real-world datasets and performing simple, reusable data automation tasks.
## Data Cleaning & Simple Automation (Python)

This repository contains small Python projects focused on cleaning real-world datasets and performing simple, reusable automation tasks.

The work demonstrates:
- cleaning and structuring messy CSV and Excel files
- handling missing values and inconsistent formats
- preparing clean, analysis-ready datasets
- writing clear and reusable Python scripts

Tools used: Python, pandas, NumPy, Jupyter Notebook.
